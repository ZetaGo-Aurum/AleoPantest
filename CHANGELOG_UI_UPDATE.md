# Changelog - Aleopantest V3.0.0
<div style="font-size: 80%; color: #666666;">by Aleocrophic</div>

## [3.2.0] - 2025-12-26

### Added
- **Dynamic Web Forms**: Implementasi sistem form dinamis yang menyesuaikan dengan kebutuhan spesifik setiap tool.
- **Custom Form Schema**: Menambahkan dukungan `form_schema` pada `ToolMetadata` untuk konfigurasi UI yang lebih fleksibel.
- **Advanced Dorking UI**: Form khusus untuk Advanced Dorking dengan pilihan Search Engine, Dork Templates, dan Custom Queries.
- **Mobile Responsive Design**: UI Web baru yang sepenuhnya responsif, mendukung penggunaan di desktop, tablet, dan smartphone.
- **Sidebar Navigation**: Navigasi kategori tool yang lebih intuitif dengan sidebar (drawer pada mobile).
- **Execution Parameters**: Dukungan untuk pengiriman parameter kustom dari Web UI ke core engine.

### Changed
- **Web UI Overhaul**: Desain ulang total menggunakan Tailwind CSS (via CDN) dengan tema dark-mode yang konsisten.
- **Web Server API**: Endpoint `/aleopantest/api/tools` kini mengembalikan metadata lengkap, dan `/aleopantest/api/run` mendukung parameter tambahan.
- **BaseTool Enhancement**: Penambahan field `form_schema` pada `ToolMetadata` dan perbaikan import typing.

### Fixed
- Perbaikan error `NameError: name 'List' is not defined` pada `base_tool.py`.
- Perbaikan isu port conflict pada web server (default port sekarang dapat dikonfigurasi).
- Perbaikan layouting yang sebelumnya kurang optimal pada perangkat mobile.
